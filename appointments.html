Appointments are mandatory for Jafza and DIP warehouses. The process varies depending on whether you are using WM9 or SCE11.

üìç Quick Routing:
  • JAFZA Shipments: Use the WM9 system.
  • DIP Shipments: Use the SCE11 system.

1. JAFZA Appointments (WM9)

  • Navigation On the left-hand sidebar, click the + sign next to Appointment, select Appointment List, and then click the New button on the top bar.
  • Initial Logistics Fill in the Traffic File Number (LOG ID), Transportation Mode, and Terms (INCO Term).
  • Carrier Selection Identify the carrier specified in Logitude (Partner section next to Freight Forwarder).

    Search for the carrier using percent signs.
    Example: For Enkay Express, search %ENKAY%, hit enter, and tick the checkbox next to the correct entry.

  • Scheduling Details
    • Start Date: Use the calendar icon to select today's date.
    • Door: Enter DOOR1, DOOR2, DOOR3, or DOOR4 (Must be capitalized).
    • Trailer Type: Always select Vehicle under 3T.
  • Terms & Insurance (INCO Logic)
    • If EXW or FOB: Set Insurance, Insurance Coverage, and Custom Duty to options containing Consignee.
    • ‚ö†Ô∏è Note: There are two "Consignee" options in the system. Always select the one shown in the reference image below.
    • Other Terms: Set options to Shipper. For Custom Duty, choose To be Paid by shipper.

    Correct Consignee Option

  • Collection Address Always set the collection slot to:
    Main WH, Plot No. S10608A, Tel. No:04-816-8000.
  • Linking the ASN Click Save. Once saved, an Add button will appear at the bottom. Click it, paste your ASN Number, hit enter, tick the correct ASN, and click OK.

    Note: Some shipments require "Appointment Only" without linking an ASN. Follow all steps but skip this final link.

2. DIP Appointments (SCE11)

  • Environment Setup Click the INFOR_ENTERPRISE dropdown at the very top and select DIP.
  • Navigation On the left menu, go to WMS -> Appointment -> Appointment List and click New.
  • Core Fields Fill out the Traffic File Number, Transportation Mode, and INCO Terms.

    The Insurance and Collection logic (Main Warehouse) is identical to the WM9 process described above.

  • Slot & Timing Fill out DOOR, Carrier, and Trailer Type (Vehicle under 3T).
    ‚è±Ô∏è Timing Rule: The End Time must be at least 10 minutes after the Start Time (Ex: 4:00 PM Start -> 4:10 PM End).
  • Save & Link Click Save, then use the Add button appearing at the bottom to link your ASN.
